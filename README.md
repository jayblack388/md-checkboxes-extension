# Markdown Checkboxes

Interactive checkboxes in Markdown preview - click to toggle `[ ]` and `[x]`.

## Features

- Renders `[ ]` as unchecked checkbox
- Renders `[x]` or `[X]` as checked checkbox
- Click to toggle checkbox state
- Automatically updates the source markdown file
- Checked items appear with strikethrough styling

## Usage

In any markdown file, use standard checkbox syntax:

```markdown
- [ ] Unchecked item
- [x] Checked item
```

Click the checkbox in the preview to toggle its state. The source file will be updated automatically.

## Installation

Search "Markdown Checkboxes" in the Extensions panel.
